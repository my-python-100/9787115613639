## 2.3　字符串

**2.3.2　在字符串中使用变量**

* 要在字符串中插入变量的值，可先在左引号前加上字母 `f`，再将要插入的变量放在`花括号`内。这样，Python 在显示字符串时，将把每个变量都替换为其值。
* 这种字符串称为 f 字符串。f 是 format（设置格式）的简写
* 还可以使用 f 字符串来创建消息，再把整条消息赋给变量
